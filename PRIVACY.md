# Privacy Policy

Last updated: June 11, 2026

ReClipt is a local macOS clipboard history and snippets app. Clipboard data
can contain sensitive information, so the app is designed to keep clipboard
history and snippets on the user's Mac.

## Summary

- ReClipt does not require a user account.
- ReClipt does not ask for names, email addresses, or account credentials.
- ReClipt does not transmit clipboard contents or snippet contents.
- ReClipt does not currently include analytics, crash reporting, or automatic
  update services in this source tree.

## Local Data

ReClipt stores app data locally, including clipboard history, snippets,
preferences, and excluded application settings.

Clipboard managers can store sensitive local data. Users should exclude password
managers and other sensitive apps from clipboard history recording when possible.

ReClipt does not currently claim that local clipboard history is encrypted.
Users who store sensitive data on their Mac should enable FileVault and use the
security controls provided by macOS.

## Network Communication

ReClipt does not intentionally use network services in this source tree.

## Analytics And Diagnostics

ReClipt does not currently include analytics or crash reporting SDKs.

## User Controls

Users can manage clipboard history behavior through app preferences, including
which pasteboard types are stored and which apps are excluded.

## Changes

This policy may be updated when app behavior changes.

## Contact

For privacy or security questions, contact the project maintainer.
